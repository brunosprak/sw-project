## How to start server

```
npm start
```

## How to run job to capture data

```
npm run batch
```

## How to run tests

```
npm test
```

## Libraries used

- Jest
- Express
- Prettier + ESLint + airbnb
- wtf_wikipedia
- "Aurek-Besh" font.
